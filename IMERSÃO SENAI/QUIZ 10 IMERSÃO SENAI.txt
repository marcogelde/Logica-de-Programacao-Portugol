4) O que é um pseudocodigo?

-Um conjunto de 0símbolos gráficos usados para criar diagramas de fluxo de dados.
-Uma forma simplificada de descrever algoritmos usando uma combinação de linguagem natural e estrutura de código.

5) Qual a diferença entre um vetor e uma matriz?
-Um vetor é uma coleção de elementos organizados em linhas e colunas, enquanto uma matriz é uma coleção de elementos organizados em uma única linha.
-Um vetor é uma estrutura de dados unidimensional que armazena uma sequência de elementos, enquanto uma matriz é uma estrutura de dados bidimensional que armazena elementos em linhas e colunas.
-Um vetor pode armazenar tipos de dados diferentes em cada posição, enquanto uma matriz só pode armazenar um único tipo de dado.

6) Qual a diferença entre uma pilha e uma fila?
-Uma pilha é como uma pilha de pratos: você tira o último que colocou primeiro. Já uma fila é como uma fila de pessoas esperando no banco, onde o primeiro que chegou é o primeiro a ser atendido.
-Uma pilha serve para organizar coisas de forma que você pegue a última que colocou primeiro, como uma torre de blocos. Uma fila é para organizar coisas em ordem, como esperar sua vez no caixa de um mercado.
-Pilhas é quando o primeiro a entrar é o primeiro a sair e fila o ultimo a entrar é o primeiro a sair
